# ✏️ 끄적끄적-Front

명지대학교 '웹 프로그래밍2' 수업 3조 "끄적끄적" 팀 프로젝트

## 시작 방법

```
# Step 0 : 처음 시작한 경우
npm install

# Step 1 : 실행
npm start
```

## 🍡 Git Commit Message

| 헤더     | 내용                                                          |
| -------- | ------------------------------------------------------------- |
| feat     | 새로운 기능 추가                                              |
| refactor | 코드 리팩토링                                                 |
| fix      | 버그 수정                                                     |
| style    | 세미 콜론 수정, 줄바꿈, css 변경 등 기능과 무관한 스타일 변경 |
| setting  | 빌드수행, 패키지 설치, 환경 설정 수정 등                      |
| docs     | 주석 추가 및 삭제, readme 작성                                |
